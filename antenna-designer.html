Method

Patch width comes from W = c / (2·fr) · √(2/(εr+1)), then the effective dielectric constant εeff is derived from the width-to-height ratio to account for the fringing fields at the patch edges.

The length extension ΔL corrects for that same fringing effect, and the final patch length is L = c / (2·fr·√εeff) − 2ΔL. Ground plane dimensions add a 6h margin on each side, which is standard practice to avoid truncating the fringing fields.

This gives you a solid starting point for simulation in tools like HFSS or CST — treat it as a first-pass estimate, not a final fabrication value.
