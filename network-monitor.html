Tool 03 — Networking

Network Health Monitor

A simplified, browser-based demo of anomaly detection on network traffic. Generate sample traffic and watch the detector flag readings that don't fit the pattern.

This runs a lightweight statistical detector (z-score against a rolling baseline) for a fast, dependency-free browser demo. The full project uses a trained Isolation Forest model — see the source on GitHub for that implementation.

Method

This browser demo computes a rolling mean and standard deviation across the traffic series, then flags any point whose z-score — |x − mean| / std — exceeds the sensitivity threshold set by the slider. It's a fast approximation that runs entirely client-side with no model to load.

The production version of this project uses Isolation Forest, an unsupervised model that isolates anomalies by how few random splits it takes to separate a point from the rest of the data — points that isolate quickly, in fewer splits, score as more anomalous. This catches multivariate and non-linear anomaly patterns that a simple z-score threshold can miss.

Full training code and the multivariate version of this model are in the GitHub repository linked from the homepage.
